# LoL Live Game Helper

League of Legends の試合中に、チームや各レーンの状況をリアルタイムで分析・表示するデスクトップアプリです。

このツールを使えば、両チームの戦力差や、どのレーンが優勢/劣勢なのかをひと目で把握でき、試合の展開を有利に進めるための判断材料として役立ちます。

## 主な機能

- **チームの戦力比較**: 両チームの合計ゴールドをリアルタイムで表示し、どちらが優勢かを示します。
- **プレイヤー情報の可視化**: 各プレイヤーのチャンピオン、レベル、装備から算出された「パワー」を表示し、個々の戦力を比較できます。
- **レーンごとの優劣分析**: 各レーン（TOP/JUNGLE/MIDDLE/BOTTOM/SUPPORT）のパワーバランスを計算し、矢印と差分で分かりやすく表示します。
- **自動更新**: 2秒ごとに最新の試合情報を自動で取得します。手動での更新も可能です。

## 使い方

1. **League of Legends の試合を開始します。**
   - 練習モード（プラクティスツール）やカスタムゲームでも利用できます。
   - **注意:** 試合が始まっていないと、データは表示されません。
2. **このアプリを起動します。**
   - 起動すると、自動的に試合データを取得し、画面に表示します。

## 画面の見方

- **上部のバー**:
  - `ORDER` (青チーム) と `CHAOS` (赤チーム) の合計ゴールドが表示されます。
  - ゴールドが多いチームが太字と矢印で強調され、どちらが経済的に有利かが分かります。
- **プレイヤーカード**:
  - 各プレイヤーのチャンピオン画像、レベル、現在の「パワー」、装備の合計ゴールドが表示されます。
  - プレイヤーはレーン順（TOP, JG, MID, BOT, SUP）に並んでいます。
- **レーン分析**:
  - チーム間に表示される中央のインジケータです。
  - 各レーンの優劣が矢印（▲▼）とパワーの差分で示されます。

## よくある質問（FAQ）

**Q. データが表示されません。**

A. League of Legends の試合（サモナーズリフトなど）が実際に始まっているか確認してください。クライアントのロビー画面やチャンピオン選択画面では、データは取得できません。それでも表示されない場合は、PCのファイアウォール設定がアプリの通信をブロックしていないか確認してください。

**Q. 「パワー」とは何ですか？**

A. 各プレイヤーの現在のレベルと所持しているアイテムの価値を基に、アプリが独自に算出した戦力の指標です。あくまで目安としてご活用ください。
